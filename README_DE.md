## Einführung

### 🚀 **Ein KI-Desktop-Begleiter mit unendlichen Möglichkeiten!**

#### Desktop-Begleiter: Unterstützt benutzerdefinierte VRM-Modelle, Aktionen, 3D-Szenen und ist kompatibel mit [live2D-Erweiterungen](https://github.com/heshengtao/sap-live2d)
![image](doc/image/img-2/cover.jpeg)

#### Link VTS: Steuern Sie Live2D-Modelle in Vtube Studio mit benutzerdefinierten Aktionen und Ausdruckssteuerung
![image](doc/image/img-2/vts.jpeg)

#### Hochflexible Chat-Oberfläche: Benutzerdefinierte Hintergrundbilder, Emoji-Pakete, Charakter-Personas, THA-basierte 2D-Charakter-Avatare und Charakter-MCP-Tools für ein immersiveres Gespräch
![image](doc/image/img-2/tha4.jpeg)

#### Aufgabencenter: Lassen Sie Ihre KI-Agenten beliebige komplexe Aufgaben im Hintergrund ausführen, steuern Sie automatisch Ihren Computer, um Arbeiten zu erledigen, mit Unterstützung für MCP und Agent Skills.
![image](doc/image/img-2/task.jpeg)

#### Computersteuerung: Durch Desktop-Vision kombiniert mit einer Toolchain zur Steuerung von Maus, Tastatur und Terminal können KI-Agenten jede Aufgabe auf einem Computer erledigen! Das Bild unten zeigt eine KI beim Spielen von Wordle.
![image](doc/image/img-2/vision.jpeg)

#### Mehrere Fenstermodi: Die Chat-Oberfläche unterstützt telefon- und kapselgroße Andockmöglichkeiten auf dem Desktop
![image](doc/image/img-1/windowmode.jpeg)

#### Multi-Rollen-Gruppenchat: Unterstützt Tavernen-Charakterkarten und Langzeitgedächtnis, sodass Sie mit mehreren Charakteren gleichzeitig chatten können!
![image](doc/image/img-2/group.jpeg)

#### Instant-Messaging-Bot: Unterstützt Ein-Klick-Bereitstellung für QQ, WeChat, Feishu, DingTalk, Telegram, Discord, Slack
![image](doc/image/img-2/im.jpeg)

#### Live-Streaming-Bot: Unterstützt Ein-Klick-Bereitstellung für Bilibili, YouTube, Twitch und ermöglicht 360-Grad-Panorama-Live-Streaming.
![image](doc/image/img-2/yt.jpeg)

#### KI-Browser: Statten Sie Ihre KI-Agenten mit einem eigenen Browser aus, der automatische Steuerung unterstützt
![image](doc/image/img-2/browser.jpeg)

#### Erweiterungssystem: Unterstützt die Installation von Erweiterungen und das Erstellen neuer. Das Bild unten zeigt eine Galgame-Erweiterung. Alle Erweiterungen können entweder in einem unabhängigen Fenster oder in einer Seitenleiste geöffnet werden. Sie können sogar die Agenten-Fertigkeit [sap-extension-creator](skills/sap-extension-creator) verwenden, um neue SAP-Erweiterungen zu erstellen und so das Bootstrap der Agenten-Party zu erreichen!
![image](doc/image/img-2/ext.jpeg)

#### Entwicklerfreundlich: Bietet offene OpenAI-API-Schnittstellen und MCP-Schnittstellen, sodass Agenten extern angebunden werden können
![image](doc/image/img-2/api.jpeg)

#### 🚀 Verwendungsschritte

**1. Netzwerk-Download-Quarantäne entfernen (Wichtig)**
Nach dem Herunterladen und Entpacken öffnen Sie das Terminal, geben Sie den folgenden Befehl ein (achten Sie auf das Leerzeichen am Ende), ziehen Sie dann den **entpackten Ordner** in das Terminal-Fenster und drücken Sie die Eingabetaste:
```shell
sudo xattr -rd com.apple.quarantine 
```
*(Hinweis: Der Parameter `-rd` entfernt rekursiv die Quarantäne-Attribute für alle Komponenten im Ordner; andernfalls funktioniert die Python-Umgebung möglicherweise nicht richtig.)*

**2. Skriptausführungsberechtigungen erteilen**
Navigieren Sie im Terminal zum Ordner und führen Sie Folgendes aus:
```shell
chmod +x 一键更新(update).sh 一键启动(start).sh
```

**3. Software ausführen**
- **Erstverwendung/Update:** Es wird empfohlen, zuerst `./一键更新(update).sh` auszuführen, um sicherzustellen, dass die Abhängigkeiten auf dem neuesten Stand sind.
- **Täglicher Start:** Führen Sie direkt `./一键启动(start).sh` aus.

## Verwendung

- Desktop: Klicken Sie auf das Desktop-Symbol, um sofort zu beginnen.

- Web oder Docker: Rufen Sie nach dem Start http://localhost:3456/ auf.

- API-Aufruf: Entwicklerfreundlich, vollständig kompatibel mit dem OpenAI-Format, Echtzeit-Ausgabe möglich, ohne die Antwortgeschwindigkeit der ursprünglichen API zu beeinträchtigen. Keine Änderung des aufrufenden Codes erforderlich:

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "Was ist Super Agent Party?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- MCP-Aufruf: Nach dem Start können Sie den lokalen MCP-Dienst aufrufen, indem Sie den folgenden Inhalt in die Konfigurationsdatei schreiben:

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp"
      }
    }
  }
  ```

- Schnellbefehle: Geben Sie in der Chat-Box oder auf einer beliebigen verbundenen IM-Plattform (Feishu, QQ, Telegram, Discord, Slack, WeChat, WeCom, DingTalk) die folgenden Befehle ein, um den Assistenten schnell zu steuern. Sie können sie in **Systemeinstellungen → Schnellbefehle** umschalten und die vollständige Liste anzeigen.

  | Befehl | Aliase | Bereich | Beschreibung |
  | --- | --- | --- | --- |
  | `/help` | `/帮助` `/?` | Chat + IM | Alle verfügbaren Schnellbefehle auflisten |
  | `/new` | `/reset` `/重启` `/新建` `/restart` | Chat + IM | Aktuelle Unterhaltung löschen und neu beginnen |
  | `/stop` | `/停止` | Chat + IM | Die aktuell generierte Antwort stoppen |
  | `/retry` | `/重试` | Nur Chat | Die letzte Antwort neu generieren |
  | `/model [provider:model]` | `/模型` | Chat + IM | Modell anzeigen oder wechseln (IM nur anzeigen) |
  | `/personality [name]` | `/角色` `/persona` | Chat + IM | Persönlichkeit anzeigen oder wechseln |
  | `/skills` | `/技能` | Chat + IM | Liste der verfügbaren Fähigkeiten durchsuchen |
  | `/id` | - | Nur IM | Aktuelle Sitzung/Channel-ChatID anzeigen |
  | `/sub` | `/订阅` | Nur IM | Aktuelle Sitzung zur proaktiven Nachrichtenliste hinzufügen |
  | `/unsub` | `/取消订阅` | Nur IM | Aktuelle Sitzung aus der proaktiven Nachrichtenliste entfernen |
  | `/plan` * | `/计划` | Chat + IM | In den Plan-Modus wechseln (schreibgeschützt) |
  | `/read` * | `/只读` | Chat + IM | In den Nur-Lese-Modus wechseln (jede Aktion bestätigen) |
  | `/edit` * | `/编辑` | Chat + IM | In den Bearbeitungsmodus wechseln |
  | `/yolo` * | - | Chat + IM | In den Yolo-Modus wechseln (volle Autonomie) |
  | `/cowork` * | `/协作` | Chat + IM | In den Cowork-Modus wechseln (Sub-Agenten-Zusammenarbeit) |
  | `/goal` * | `/目标` | Chat + IM | In den Ziel-Modus wechseln (autonome Schleife) |
  | `/<skill-name>` * | - | Chat + IM | Die angegebene Fertigkeit einfügen |
  | `#<text>` * | - | Chat + IM | Arbeitsbereich-Speicher speichern |
  | `@<path>` * | - | Nur Chat | Eine Arbeitsbereichsdatei referenzieren |

  > `*` Erfordert aktivierte Computer-CLI-Steuerung; Fertigkeitseinspritzung, `#`-Speicher und `@`-Dateireferenz erfordern ebenfalls einen konfigurierten Arbeitsbereich.
