## Introducción

### 🚀 **¡Un compañero de escritorio con IA con infinitas posibilidades!**

#### Compañero de escritorio: Soporta modelos VRM personalizados, acciones, escenas 3D y es compatible con [extensiones live2D](https://github.com/heshengtao/sap-live2d)
![image](doc/image/img-2/cover.jpeg)

#### Link VTS: Controla modelos Live2D en Vtube Studio, con soporte para acciones personalizadas y control de expresiones
![image](doc/image/img-2/vts.jpeg)

#### Interfaz de chat de alta libertad: fondos de pantalla personalizados, paquetes de emojis, personajes, avatares 2D basados en THA y herramientas MCP de personaje para una conversación más inmersiva
![image](doc/image/img-2/tha4.jpeg)

#### Centro de tareas: Permite que tus agentes de IA realicen cualquier tarea avanzada en segundo plano, controlando automáticamente tu computadora para completar el trabajo, con soporte para MCP y Agent Skills.
![image](doc/image/img-2/task.jpeg)

#### Control de computadora: A través de la visión del escritorio combinada con un conjunto de herramientas para controlar el ratón, el teclado y la terminal, los agentes de IA pueden realizar cualquier tarea en una computadora. La imagen de abajo muestra a una IA jugando Wordle.
![image](doc/image/img-2/vision.jpeg)

#### Múltiples modos de ventana: La interfaz de chat admite acoplamiento en tamaño de teléfono y tamaño de cápsula en el escritorio
![image](doc/image/img-1/windowmode.jpeg)

#### Chat grupal con múltiples roles: Compatible con tarjetas de personaje tavern y memoria a largo plazo, ¡permitiéndote chatear con múltiples personajes simultáneamente!
![image](doc/image/img-2/group.jpeg)

#### Bot de mensajería instantánea: Soporta despliegue con un clic en QQ, WeChat, Feishu, DingTalk, Telegram, Discord, Slack
![image](doc/image/img-2/im.jpeg)

#### Bot de transmisión en vivo: Soporta despliegue con un clic en Bilibili, YouTube, Twitch y permite transmisión en vivo panorámica de 360 grados.
![image](doc/image/img-2/yt.jpeg)

#### Navegador IA: Equipa a tus agentes de IA con su propio navegador, compatible con control automático
![image](doc/image/img-2/browser.jpeg)

#### Sistema de extensiones: Soporta la instalación y creación de nuevas extensiones. La imagen de abajo muestra una extensión de galgame. Todas las extensiones se pueden abrir en una ventana independiente o en una barra lateral. Incluso puedes usar la habilidad de agente [sap-extension-creator](skills/sap-extension-creator) para crear nuevas extensiones SAP, ¡logrando el bootstrap de la agent party!
![image](doc/image/img-2/ext.jpeg)

#### Amigable para desarrolladores: Ofrece interfaces de API OpenAI y MCP abiertas, permitiendo que los agentes se conecten externamente
![image](doc/image/img-2/api.jpeg)

> **Adecuado para desarrolladores/usuarios avanzados**: Versión de código fuente sin instalación, soporta sincronización con un clic a la última versión del repositorio, sin necesidad de esperar al empaquetado de la versión de escritorio.

👉 [Descargar](https://github.com/heshengtao/super-agent-party/releases/download/v0.4.2/super-agent-party-mac-v0.4.2.zip)

#### 🚀 Pasos de uso

**1. Eliminar la cuarentena de descarga de red (Importante)**
Después de descargar y extraer, abre la Terminal, ingresa el siguiente comando (nota que hay un espacio al final), luego arrastra la **carpeta extraída** a la ventana de Terminal y presiona Enter:
```shell
sudo xattr -rd com.apple.quarantine 
```
*(Nota: El parámetro `-rd` elimina recursivamente los atributos de cuarentena de todos los componentes dentro de la carpeta; de lo contrario, el entorno Python podría no funcionar correctamente).*

**2. Otorgar permisos de ejecución de scripts**
Navega a la carpeta en la Terminal y ejecuta:
```shell
chmod +x 一键更新(update).sh 一键启动(start).sh
```

**3. Ejecutar el software**
- **Primer uso/Actualización:** Se recomienda ejecutar primero `./一键更新(update).sh` para asegurar que las dependencias estén sincronizadas con la última versión.
- **Inicio diario:** Ejecuta directamente `./一键启动(start).sh`.

### Instalación de escritorio para macOS (Actualmente solo compatible con chips M)

  👉 [Descargar](https://github.com/heshengtao/super-agent-party/releases/download/v0.4.2/Super-Agent-Party-0.4.2-Mac.dmg)

⭐ ¡Atención! Después de descargar, arrastra el archivo de aplicación del dmg al directorio `/Applications`, luego abre la terminal, ejecuta el siguiente comando e ingresa la contraseña de root para eliminar el atributo de cuarentena de la descarga de red:

  ```shell
  sudo xattr -dr com.apple.quarantine /Applications/Super-Agent-Party.app
  ```

```shell
docker pull ailm32442/super-agent-party:latest
docker run -d -p 3456:3456 -v ./super-agent-data:/app/data ailm32442/super-agent-party:latest
```

⭐ ¡Nota! `./super-agent-data` se puede reemplazar con cualquier carpeta local. Después de iniciar Docker, todos los datos se almacenarán en esta carpeta local y no se cargarán en ningún otro lugar.

⭐ ¡Nota! El nombre de usuario predeterminado es `root` y la contraseña predeterminada es `pass`. Cambia la contraseña después del primer inicio de sesión.

## Requisitos del sistema

- CPU: 2 núcleos o más
- Memoria: 2GB o más

**Debido a que todos los modelos son opcionales, puedes acceder al motor de despliegue local, o puedes usar exclusivamente interfaces de proveedores de servicios en la nube, por lo que los requisitos de hardware son mínimos. La versión Docker se probó en un servidor en la nube de 2 núcleos y 2 GB y funciona sin problemas.**

## Uso

- Escritorio: Haz clic en el icono del escritorio para usarlo de inmediato.

- Web o Docker: Accede a http://localhost:3456/ después de iniciar.

- Llamada a la API: Amigable para desarrolladores, totalmente compatible con el formato OpenAI, puede generar en tiempo real y no afecta la velocidad de respuesta de la API original. No es necesario modificar el código de llamada:

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "¿Qué es Super Agent Party?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- Llamada MCP: Después de iniciar, puedes invocar el servicio MCP local escribiendo el siguiente contenido en el archivo de configuración:

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp",
      }
    }
  }
  ```

- Comandos rápidos: En la caja de chat o en cualquier plataforma de mensajería conectada (Feishu, QQ, Telegram, Discord, Slack, WeChat, WeCom, DingTalk), escribe los siguientes comandos para controlar rápidamente al asistente. Puedes activarlos/desactivarlos y ver la lista completa en **Configuración del sistema → Comandos rápidos**.

  | Comando | Alias | Ámbito | Descripción |
  | --- | --- | --- | --- |
  | `/help` | `/帮助` `/?` | Chat + IM | Lista todos los comandos rápidos disponibles |
  | `/new` | `/reset` `/重启` `/新建` `/restart` | Chat + IM | Borra la conversación actual y comienza de nuevo |
  | `/stop` | `/停止` | Chat + IM | Detiene la respuesta que se está generando |
  | `/retry` | `/重试` | Solo chat | Regenera la última respuesta |
  | `/model [provider:model]` | `/模型` | Chat + IM | Muestra o cambia el modelo (IM solo muestra) |
  | `/personality [name]` | `/角色` `/persona` | Chat + IM | Muestra o cambia la personalidad |
  | `/skills` | `/技能` | Chat + IM | Navega por la lista de habilidades disponibles |
  | `/id` | - | Solo IM | Muestra el ChatID de la sesión/canal actual |
  | `/sub` | `/订阅` | Solo IM | Añade la sesión actual a la lista de mensajes proactivos |
  | `/unsub` | `/取消订阅` | Solo IM | Elimina la sesión actual de la lista de mensajes proactivos |
  | `/plan` * | `/计划` | Chat + IM | Cambia al modo plan (solo lectura) |
  | `/read` * | `/只读` | Chat + IM | Cambia al modo solo lectura (confirmar cada acción) |
  | `/edit` * | `/编辑` | Chat + IM | Cambia al modo de aceptar ediciones |
  | `/yolo` * | - | Chat + IM | Cambia al modo Yolo (autonomía total) |
  | `/cowork` * | `/协作` | Chat + IM | Cambia al modo de colaboración (sub-agentes) |
  | `/goal` * | `/目标` | Chat + IM | Cambia al modo objetivo (bucle autónomo) |
  | `/<skill-name>` * | - | Chat + IM | Inyecta la habilidad especificada |
  | `#<text>` * | - | Chat + IM | Guarda memoria del espacio de trabajo |
  | `@<path>` * | - | Solo chat | Referencia un archivo del espacio de trabajo |

  > `*` Requiere que el control CLI de la computadora esté habilitado; la inyección de habilidades, la memoria `#` y la referencia de archivos `@` también requieren un espacio de trabajo configurado.
