#### 🚀 خطوات الاستخدام

**1. إزالة الحجر الصحي لتنزيلات الشبكة (هام)**
بعد التنزيل والفك، افتح الطرفية، أدخل الأمر التالي (لاحظ وجود مسافة في النهاية)، ثم اسحب **المجلد الذي تم فكه** إلى نافذة الطرفية واضغط Enter:
```shell
sudo xattr -rd com.apple.quarantine 
```
*(ملاحظة: المعامل `-rd` يزيل سمات الحجر الصحي بشكل متكرر لجميع المكونات داخل المجلد؛ وإلا فقد لا تعمل بيئة Python بشكل صحيح.)*

**2. منح صلاحيات تنفيذ البرنامج النصي**
انتقل إلى المجلد في الطرفية ونفذ:
```shell
chmod +x 一键更新(update).sh 一键启动(start).sh
```

**3. تشغيل البرنامج**
- **الاستخدام الأول/التحديث:** يُوصى بتنفيذ `./一键更新(update).sh` أولاً لضمان مزامنة التبعيات إلى الأحدث.
- **بدء التشغيل اليومي:** قم بتنفيذ `./一键启动(start).sh` مباشرة.

## الاستخدام

- سطح المكتب: انقر على أيقونة سطح المكتب للاستخدام الفوري.

- الويب أو docker: الوصول إلى http://localhost:3456/ بعد التشغيل.

- استدعاء API: صديق للمطورين، متوافق تمامًا مع تنسيق OpenAI، يمكنه الإخراج في الوقت الفعلي، ولا يؤثر على سرعة استجابة API الأصلية. لا حاجة لتعديل كود الاستدعاء:

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "What is Super Agent Party?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- استدعاء MCP: بعد التشغيل، يمكنك استدعاء خدمة MCP المحلية عن طريق كتابة المحتوى التالي في ملف التكوين:

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp",
      }
    }
  }
  ```

- أوامر الاختصار: في مربع الدردشة أو أي منصة مراسلة فورية متصلة (Feishu, QQ, Telegram, Discord, Slack, WeChat, WeCom, DingTalk)، اكتب الأوامر أدناه للتحكم السريع في المساعد. يمكنك تبديلها وعرض القائمة الكاملة في **System Settings → Shortcut Commands**.

  | الأمر | الأسماء المستعارة | النطاق | الوصف |
  | --- | --- | --- | --- |
  | `/help` | `/帮助` `/?` | الدردشة + IM | عرض جميع أوامر الاختصار المتاحة |
  | `/new` | `/reset` `/重启` `/新建` `/restart` | الدردشة + IM | مسح المحادثة الحالية والبدء من جديد |
  | `/stop` | `/停止` | الدردشة + IM | إيقاف الرد الذي يتم إنشاؤه حاليًا |
  | `/retry` | `/重试` | الدردشة فقط | إعادة إنشاء آخر رد |
  | `/model [provider:model]` | `/模型` | الدردشة + IM | عرض أو تبديل النموذج (IM يعرض فقط) |
  | `/personality [name]` | `/角色` `/persona` | الدردشة + IM | عرض أو تبديل الشخصية |
  | `/skills` | `/技能` | الدردشة + IM | تصفح قائمة المهارات المتاحة |
  | `/id` | - | IM فقط | عرض ChatID للجلسة/القناة الحالية |
  | `/sub` | `/订阅` | IM فقط | إضافة الجلسة الحالية إلى قائمة الرسائل الاستباقية |
  | `/unsub` | `/取消订阅` | IM فقط | إزالة الجلسة الحالية من قائمة الرسائل الاستباقية |
  | `/plan` * | `/计划` | الدردشة + IM | التبديل إلى وضع التخطيط (للقراءة فقط) |
  | `/read` * | `/只读` | الدردشة + IM | التبديل إلى وضع القراءة فقط (تأكيد كل إجراء) |
  | `/edit` * | `/编辑` | الدردشة + IM | التبديل إلى وضع قبول التعديلات |
  | `/yolo` * | - | الدردشة + IM | التبديل إلى وضع Yolo (استقلالية كاملة) |
  | `/cowork` * | `/协作` | الدردشة + IM | التبديل إلى وضع العمل المشترك (تعاون الوكيل الفرعي) |
  | `/goal` * | `/目标` | الدردشة + IM | التبديل إلى وضع الهدف (حلقة مستقلة) |
  | `/<skill-name>` * | - | الدردشة + IM | حقن المهارة المحددة |
  | `#<text>` * | - | الدردشة + IM | حفظ ذاكرة مساحة العمل |
  | `@<path>` * | - | الدردشة فقط | الإشارة إلى ملف في مساحة العمل |

  > `*` يتطلب تمكين التحكم في CLI للكمبيوتر؛ حقن المهارة، وذاكرة `#` والإشارة إلى الملف `@` تتطلب أيضًا مساحة عمل مهيأة.

## ملاحظات خاصة
1. بعض ميزات هذا المشروع مفتوح المصدر (مثل تركيب الصوت Edge TTS، إلخ) تعتمد على واجهات عامة أو وظائف تجريبية تقدمها خدمات طرف ثالث. قد تصبح هذه الميزات غير متاحة في أي وقت بسبب تغييرات في سياسات الطرف الثالث. لا يتحمل المطور أي مسؤولية عن استقرارها أو قانونيتها أو استمراريتها. باستخدام هذا المشروع، يعتبر المستخدمون قد فهموا ووافقوا على تحمل المخاطر المرتبطة. لا يوصي المطور أو يشجع على استخدام هذه الوظائف في السيناريوهات التجارية أو واسعة النطاق.

2. يستخدم بوت QQ واجهة بوت QQ الرسمية. يُرجى الامتثال لـ [إرشادات استخدام بوت QQ AIGC](https://q.qq.com/#/news/detail?id=1376238e8e2fbbc036676bb09d2f37da).

3. وظيفة التحكم في المتصفح المقدمة في هذا المشروع هي واجهة تصفح مدعومة بإمكانية الوصول تعتمد على نماذج اللغة الكبيرة (LLMs). وهي مصممة لمساعدة الأشخاص ذوي الإعاقة البصرية وكبار السن أو أولئك الذين يعانون من تحديات في الحركة على تشغيل المتصفحات بسهولة أكبر من خلال أوامر اللغة الطبيعية، باستخدام تقنية التعرف البصري بالذكاء الاصطناعي. وهي ليست مخصصة للزحف الآلي أو القرصنة. يعتمد المشروع على بنية تقنية "الاستدلال البصري LLM → تشغيل خطوة واحدة". تتميز واجهة التصفح المدعومة بإمكانية الوصول بالخصائص التالية:
   أ. عدم التزامن عالي التردد: بسبب الاعتماد على سرعة استدلال LLM (3-5 ثوانٍ لكل خطوة) وخوارزميات التأخير العشوائي الشبيهة بالبشر المضمنة، فإن تكرار تشغيل الأداة أقل تمامًا من السرعة اليدوية القصوى للمستخدم البشري العادي.
   ب. لا ضغط على الخادم: لا تدعم هذه الأداة التزامن متعدد الخيوط أو تجريف البيانات الجماعي أو هجمات DDoS. من منظور الخادم، لا يمكن تمييز سلوكها عن سلوك المستخدم البشري العادي ولن تفرض عبئًا إضافيًا على خادم الموقع المستهدف.

4. لا تستخدم هذا المشروع على صفحات البنوك أو بوابات الدفع أو المعلومات شديدة السرية. المطور غير مسؤول عن خروقات الخصوصية الناتجة عن عمليات المستخدم غير الصحيحة. تشمل السلوكيات المحظورة تجريف البيانات على نطاق واسع، وتجاوز آليات الأمان، والتدخل في الشبكة، وانتهاكات القوانين واللوائح.

5. هذا المشروع هو أداة مفتوحة المصدر تم تطويرها بشكل مستقل. عند استخدام المستخدمين لهذا البرنامج للوصول إلى خدمات API تابعة لجهات خارجية، فهم مسؤولون عن الامتثال لشروط الخدمة ذات الصلة لمزودي الخدمة.

6. أي محتوى يتم إنشاؤه بواسطة هذا البرنامج من خلال نماذج كبيرة تابعة لجهات خارجية هو مسؤولية مزودي النموذج وإجراءات المستخدم من حيث الدقة والاكتمال والامتثال. لا يتحمل مؤلف هذا البرنامج أي مسؤولية قانونية عن هذا المحتوى.
