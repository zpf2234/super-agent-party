#### 🚀 사용 방법

**1. 네트워크 다운로드 격리 제거 (중요)**
다운로드 및 압축 해제 후, 터미널을 열고 다음 명령어를 입력한 후 (끝에 공백이 있음에 주의), **압축 해제된 폴더**를 터미널 창으로 드래그한 다음 Enter 키를 누르세요:
```shell
sudo xattr -rd com.apple.quarantine 
```
*(참고: `-rd` 매개변수는 폴더 내 모든 구성 요소의 격리 속성을 재귀적으로 제거합니다. 그렇지 않으면 Python 환경이 정상적으로 작동하지 않을 수 있습니다.)*

**2. 스크립트 실행 권한 부여**
터미널에서 해당 폴더로 이동하여 다음을 실행하세요:
```shell
chmod +x 一键更新(update).sh 一键启动(start).sh
```

**3. 소프트웨어 실행**
- **처음 사용/업데이트:** 먼저 `./一键更新(update).sh`를 실행하여 종속성이 최신 버전으로 동기화되었는지 확인하는 것이 좋습니다.
- **일상적인 시작:** 직접 `./一键启动(start).sh`를 실행하세요.

## 하드웨어 요구 사항

- CPU: 2코어 이상
- 메모리: 2GB 이상

**모든 모델은 선택 사항이며, 로컬 배포 엔진에 접속하거나 전부 클라우드 서비스 제공자 인터페이스를 사용할 수 있으므로 하드웨어 요구 사항이 거의 없습니다. 2코어 2GB 클라우드 서버에서 Docker 버전을 테스트했으며, 정상적으로 작동하는 것을 확인했습니다.**

- 데스크톱: 데스크톱 아이콘을 클릭하여 즉시 사용할 수 있습니다.

- 웹 또는 Docker: 시작 후 http://localhost:3456/ 에 접속하세요.

- API 호출: 개발자 친화적이며, OpenAI 형식과 완벽하게 호환됩니다. 실시간으로 출력 가능하며, 원본 API의 응답 속도에 영향을 주지 않습니다. 호출 코드를 수정할 필요가 없습니다:

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "Super Agent Party란 무엇인가요?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- MCP 호출: 시작 후 설정 파일에 다음 내용을 작성하여 로컬 MCP 서비스를 호출할 수 있습니다:

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp"
      }
    }
  }
  ```
