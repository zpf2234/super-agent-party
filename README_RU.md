#### 🚀 Инструкция по использованию

**1. Снятие карантина сетевой загрузки (Важно)**
После загрузки и распаковки откройте Терминал, введите следующую команду (обратите внимание на пробел в конце), затем перетащите **распакованную папку** в окно Терминала и нажмите Enter:
```shell
sudo xattr -rd com.apple.quarantine 
```
*(Примечание: параметр `-rd` рекурсивно удаляет атрибуты карантина для всех компонентов внутри папки; в противном случае окружение Python может работать некорректно.)*

**2. Предоставление прав на выполнение скриптов**
В Терминале перейдите в папку и выполните:
```shell
chmod +x 一键更新(update).sh 一键启动(start).sh
```

**3. Запуск**
- **Первое использование/Обновление:** рекомендуется сначала выполнить `./一键更新(update).sh`, чтобы синхронизировать зависимости до последней версии.
- **Ежедневный запуск:** непосредственно выполните `./一键启动(start).sh`.

## Системные требования

- CPU: 2 ядра или более
- Память: 2 ГБ или более

**Поскольку все модели являются опциональными, вы можете использовать локальный движок развёртывания или интерфейсы облачных провайдеров, поэтому аппаратные требования минимальны. Docker-версия была протестирована на 2-ядерном 2-гигабайтном облачном сервере и работает отлично.**

## Использование

- Десктоп: нажмите на иконку на рабочем столе для мгновенного использования.

- Веб или Docker: после запуска перейдите по адресу http://localhost:3456/.

- API-вызов: дружелюбно к разработчикам, полностью совместимо с форматом OpenAI, поддерживает вывод в реальном времени, не влияя на скорость ответа оригинального API. Нет необходимости изменять код вызова:

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "Что такое Super Agent Party?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- MCP-вызов: после запуска вы можете вызвать локальный MCP-сервис, записав следующее содержимое в файл конфигурации:

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp",
      }
    }
  }
  ```

- Команды быстрого доступа: в поле ввода чата или в любой подключённой IM-платформе (Feishu, QQ, Telegram, Discord, Slack, WeChat, WeCom, DingTalk) введите указанные ниже команды для быстрого управления ассистентом. Вы можете включать/отключать их и просматривать полный список в **Настройки системы → Команды быстрого доступа**.

  | Команда | Псевдонимы | Область | Описание |
  | --- | --- | --- | --- |
  | `/help` | `/帮助` `/?` | Чат + IM | Показать список всех доступных команд быстрого доступа |
  | `/new` | `/reset` `/重启` `/新建` `/restart` | Чат + IM | Очистить текущий разговор и начать новый |
  | `/stop` | `/停止` | Чат + IM | Остановить текущую генерацию ответа |
  | `/retry` | `/重试` | Только чат | Повторно сгенерировать последний ответ |
  | `/model [provider:model]` | `/模型` | Чат + IM | Показать или сменить модель (в IM только отображение) |
  | `/personality [name]` | `/角色` `/persona` | Чат + IM | Показать или сменить личность |
  | `/skills` | `/技能` | Чат + IM | Просмотреть список доступных навыков |
  | `/id` | - | Только IM | Показать ChatID текущей сессии/канала |
  | `/sub` | `/订阅` | Только IM | Добавить текущую сессию в список проактивных сообщений |
  | `/unsub` | `/取消订阅` | Только IM | Удалить текущую сессию из списка проактивных сообщений |
  | `/plan` * | `/计划` | Чат + IM | Переключиться в режим планирования (только чтение) |
  | `/read` * | `/只读` | Чат + IM | Переключиться в режим только чтения (подтверждать каждое действие) |
  | `/edit` * | `/编辑` | Чат + IM | Переключиться в режим принятия правок |
  | `/yolo` * | - | Чат + IM | Переключиться в режим Yolo (полная автономия) |
  | `/cowork` * | `/协作` | Чат + IM | Переключиться в режим совместной работы (взаимодействие суб-агентов) |
  | `/goal` * | `/目标` | Чат + IM | Переключиться в режим цели (автономный цикл) |
  | `/<skill-name>` * | - | Чат + IM | Внедрить указанный навык |
  | `#<text>` * | - | Чат + IM | Сохранить память рабочего пространства |
  | `@<path>` * | - | Только чат | Сослаться на файл в рабочем пространстве |

  > `*` Требуется включение управления через CLI компьютера; внедрение навыков, `#` память и `@` ссылки на файлы также требуют настроенного рабочего пространства.
