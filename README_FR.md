## Introduction

### 🚀 **Un compagnon de bureau IA aux possibilités infinies !**

#### Compagnon de bureau : Prend en charge les modèles VRM personnalisés, les actions, les scènes 3D, et est compatible avec les [extensions live2D](https://github.com/heshengtao/sap-live2d)
![image](doc/image/img-2/cover.jpeg)

#### Link VTS : Contrôlez les modèles Live2D dans Vtube Studio, avec prise en charge des actions personnalisées et du contrôle des expressions
![image](doc/image/img-2/vts.jpeg)

#### Interface de chat hautement personnalisable : images d'arrière-plan personnalisées, packs d'émojis, personnalités de personnages, avatars de personnages 2D basés sur THA et outils MCP de personnages pour une conversation plus immersive
![image](doc/image/img-2/tha4.jpeg)

#### Centre de tâches : Permettez à vos agents IA d'effectuer toutes les tâches avancées en arrière-plan, de contrôler automatiquement votre ordinateur pour faire le travail, avec prise en charge de MCP et des compétences d'agent.
![image](doc/image/img-2/task.jpeg)

#### Contrôle de l'ordinateur : Grâce à la vision du bureau combinée à une chaîne d'outils pour contrôler la souris, le clavier et le terminal, les agents IA peuvent accomplir n'importe quelle tâche sur un ordinateur ! L'image ci-dessous montre une IA jouant à Wordle.
![image](doc/image/img-2/vision.jpeg)

#### Modes de fenêtre multiples : L'interface de chat prend en charge les tailles téléphone et capsule pour s'ancrer sur le bureau
![image](doc/image/img-1/windowmode.jpeg)

#### Discussion de groupe multi-rôles : Prend en charge les cartes de personnages tavern et la mémoire à long terme, vous permettant de discuter avec plusieurs personnages simultanément !
![image](doc/image/img-2/group.jpeg)

#### Bot de messagerie instantanée : Prend en charge le déploiement en un clic sur QQ, WeChat, Feishu, DingTalk, Telegram, Discord, Slack
![image](doc/image/img-2/im.jpeg)

#### Bot de diffusion en direct : Prend en charge le déploiement en un clic sur Bilibili, YouTube, Twitch, et permet la diffusion en direct panoramique à 360 degrés.
![image](doc/image/img-2/yt.jpeg)

#### Navigateur IA : Équipez vos agents IA de leur propre navigateur, avec prise en charge du contrôle automatique
![image](doc/image/img-2/browser.jpeg)

#### Système d'extensions : Prend en charge l'installation et la création de nouvelles extensions. L'image ci-dessous montre une extension galgame. Toutes les extensions peuvent être ouvertes soit dans une fenêtre indépendante, soit dans un panneau latéral. Vous pouvez même utiliser la compétence d'agent [sap-extension-creator](skills/sap-extension-creator) pour créer de nouvelles extensions SAP, réalisant ainsi l'auto-amorçage de la fête d'agents !
![image](doc/image/img-2/ext.jpeg)

#### Convivial pour les développeurs : Offre des interfaces API OpenAI ouvertes et des interfaces MCP, permettant aux agents d'être connectés de l'extérieur
![image](doc/image/img-2/api.jpeg)

#### 🚀 Étapes d'utilisation

**1. Supprimer la quarantaine de téléchargement réseau (Important)**
Après avoir téléchargé et extrait, ouvrez le Terminal, entrez la commande suivante (notez qu'il y a un espace à la fin), puis faites glisser le **dossier extrait** dans la fenêtre du Terminal et appuyez sur Entrée :
```shell
sudo xattr -rd com.apple.quarantine 
```
*(Remarque : Le paramètre `-rd` supprime récursivement les attributs de quarantaine pour tous les composants du dossier ; sinon, l'environnement Python pourrait ne pas fonctionner correctement.)*

**2. Accorder les permissions d'exécution des scripts**
Accédez au dossier dans le Terminal et exécutez :
```shell
chmod +x 一键更新(update).sh 一键启动(start).sh
```

**3. Lancer le logiciel**
- **Première utilisation/Mise à jour :** Il est recommandé d'exécuter d'abord `./一键更新(update).sh` pour s'assurer que les dépendances sont synchronisées avec la dernière version.
- **Démarrage quotidien :** Exécutez directement `./一键启动(start).sh`.

### Installation bureau macOS (Actuellement uniquement compatible avec les puces M)

  👉 [Télécharger](https://github.com/heshengtao/super-agent-party/releases/download/v0.4.2/Super-Agent-Party-0.4.2-Mac.dmg)

⭐ Attention ! Après le téléchargement, faites glisser l'application du fichier dmg dans le répertoire `/Applications`, puis ouvrez le terminal, exécutez la commande suivante et entrez le mot de passe root pour supprimer l'attribut de quarantaine attaché au téléchargement réseau :

  ```shell
  sudo xattr -dr com.apple.quarantine /Applications/Super-Agent-Party.app
  ```

### Déploiement Docker (Cette version de l'animal de bureau n'est visible que via un navigateur)

```shell
docker pull ailm32442/super-agent-party:latest
docker run -d -p 3456:3456 -v ./super-agent-data:/app/data ailm32442/super-agent-party:latest
```

⭐ Note ! `./super-agent-data` peut être remplacé par n'importe quel dossier local. Après le démarrage de Docker, toutes les données seront mises en cache dans ce dossier local et ne seront téléchargées nulle part.

**Prêt à l'emploi :** Accédez à http://localhost:3456/

### Déploiement Docker Compose (Cette version de l'animal de bureau n'est visible que via un navigateur, et un conteneur passerelle supplémentaire sera démarré pour la gestion des connexions)

**Prêt à l'emploi :** Accédez à http://localhost:3456/

**Gestion des clés API :** Accédez à http://localhost:3456/token.html

## Configuration requise

- CPU : 2 cœurs ou plus
- Mémoire : 2 Go ou plus

**Étant donné que tous les modèles sont facultatifs, vous pouvez accéder au moteur de déploiement local, ou utiliser uniquement des interfaces de fournisseurs de services cloud, donc les exigences matérielles sont minimes. La version Docker a été testée sur un serveur cloud à 2 cœurs et 2 Go et fonctionne parfaitement.**

## Utilisation

- Bureau : Cliquez sur l'icône du bureau pour utiliser immédiatement.

- Web ou Docker : Accédez à http://localhost:3456/ après le démarrage.

- Appel API : Convivial pour les développeurs, parfaitement compatible avec le format OpenAI, peut produire en temps réel et n'affecte pas la vitesse de réponse de l'API d'origine. Aucune modification du code d'appel n'est nécessaire :

  ```python
  from openai import OpenAI
  client = OpenAI(
    api_key="super-secret-key",
    base_url="http://localhost:3456/v1"
  )
  response = client.chat.completions.create(
    model="super-model",
    messages=[
        {"role": "user", "content": "Qu'est-ce que Super Agent Party ?"}
    ]
  )
  print(response.choices[0].message.content)
  ```

- Appel MCP : Après le démarrage, vous pouvez invoquer le service MCP local en écrivant le contenu suivant dans le fichier de configuration :

  ```json
  {
    "mcpServers": {
      "super-agent-party": {
        "url": "http://127.0.0.1:3456/mcp",
      }
    }
  }
  ```

- Commandes rapides : Dans la boîte de chat ou sur toute plateforme de messagerie connectée (Feishu, QQ, Telegram, Discord, Slack, WeChat, WeCom, DingTalk), tapez les commandes ci-dessous pour contrôler rapidement l'assistant. Vous pouvez les activer/désactiver et voir la liste complète dans **Paramètres système → Commandes rapides**.

  | Commande | Alias | Portée | Description |
  | --- | --- | --- | --- |
  | `/help` | `/帮助` `/?` | Chat + IM | Lister toutes les commandes rapides disponibles |
  | `/new` | `/reset` `/重启` `/新建` `/restart` | Chat + IM | Effacer la conversation en cours et recommencer |
  | `/stop` | `/停止` | Chat + IM | Arrêter la réponse en cours de génération |
  | `/retry` | `/重试` | Chat uniquement | Régénérer la dernière réponse |
  | `/model [provider:model]` | `/模型` | Chat + IM | Afficher ou changer le modèle (IM affiche seulement) |
  | `/personality [name]` | `/角色` `/persona` | Chat + IM | Afficher ou changer la personnalité |
  | `/skills` | `/技能` | Chat + IM | Parcourir la liste des compétences disponibles |
  | `/id` | - | IM uniquement | Afficher l'ID de session/channel actuel |
  | `/sub` | `/订阅` | IM uniquement | Ajouter la session actuelle à la liste des messages proactifs |
  | `/unsub` | `/取消订阅` | IM uniquement | Retirer la session actuelle de la liste des messages proactifs |
  | `/plan` * | `/计划` | Chat + IM | Passer en mode plan (lecture seule) |
  | `/read` * | `/只读` | Chat + IM | Passer en mode lecture seule (confirmer chaque action) |
  | `/edit` * | `/编辑` | Chat + IM | Passer en mode acceptation des modifications |
  | `/yolo` * | - | Chat + IM | Passer en mode Yolo (autonomie complète) |
  | `/cowork` * | `/协作` | Chat + IM | Passer en mode collaboration (sous-agents) |
  | `/goal` * | `/目标` | Chat + IM | Passer en mode objectif (boucle autonome) |
  | `/<skill-name>` * | - | Chat + IM | Injecter la compétence spécifiée |
  | `#<text>` * | - | Chat + IM | Sauvegarder la mémoire de l'espace de travail |
  | `@<path>` * | - | Chat uniquement | Référencer un fichier de l'espace de travail |

  > `*` Nécessite que le contrôle CLI de l'ordinateur soit activé ; l'injection de compétences, la mémoire `#` et la référence de fichier `@` nécessitent également un espace de travail configuré.

3. La fonctionnalité de contrôle du navigateur fournie dans ce projet est une interface de navigation assistée basée sur les grands modèles de langage (LLM). Elle est conçue pour aider les personnes malvoyantes, les personnes âgées ou celles ayant des difficultés motrices à utiliser le navigateur plus facilement grâce à des commandes en langage naturel, en utilisant la technologie de reconnaissance visuelle IA. Elle n'est pas destinée au crawl automatisé ou au piratage. Le projet adopte une architecture technique de « raisonnement visuel LLM → opération en une seule étape ». L'interface de navigation assistée présente les caractéristiques suivantes :
   a. Non-concurrence à haute fréquence : En raison de la dépendance à la vitesse de raisonnement du LLM (3 à 5 secondes par étape) et d'algorithmes de délai aléatoire intégrés imitant le comportement humain, la fréquence d'opération de l'outil est strictement inférieure à la vitesse manuelle maximale d'un utilisateur humain typique.
   b. Aucune pression sur le serveur : Cet outil ne prend pas en charge la concurrence multi-thread, l'extraction de données par lots ou les attaques DDoS. Du point de vue du serveur, son comportement est indistinguable de celui d'un utilisateur humain ordinaire et n'imposera aucune charge supplémentaire au serveur du site cible.
